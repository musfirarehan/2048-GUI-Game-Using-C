# 2048 GUI Game
A graphical implementation of the popular 2048 puzzle game developed in C.
## Project Overview
This project was developed as part of my first semester coursework. The objective was to create a fully functional version of the 2048 game with a graphical user interface.
## Features
- Interactive GUI
- Tile merging mechanics
- Size Customization
- Score tracking
- Theme Customization
- Keyboard controls
## Technologies Used
- C Programming Language
- Raylib
